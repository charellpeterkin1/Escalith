# Escalith Benchmark v0.1

Version 0.1 is a manually curated benchmark designed to validate Escalith's taxonomy and annotation methodology before larger-scale synthetic data generation.

The benchmark contains:

- 10 conversations
- 4–6 turns each
- Benign controls
- Escalating misuse examples
- Human-written annotations

Future versions will expand category coverage and introduce adversarial paraphrases.
