# Escalith Annotation Guide

Each conversation will be labeled by turn.

Every turn receives:

- Conversation ID
- Turn Number
- User Message
- Category
- Risk Level
- Escalation Stage
- Recommended Action
- Notes

Example

Conversation: 0001

Turn 1

Category:
Cybersecurity Education

Risk:
1

Action:
Allow

Reason:
General educational question.

Turn 2

Category:
Cybersecurity Education

Risk:
2

Action:
Clarify

Reason:
Conversation becoming more specific.

Turn 3

Category:
Social Engineering

Risk:
4

Action:
Refuse

Reason:
User requests actionable phishing content.
